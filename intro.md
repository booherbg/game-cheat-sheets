Quick-reference rules for card games—Euchre, Hearts, Spades, Whist, Pinochle, and a few solitaire games—written in Markdown and built to HTML so they’re easy to browse and print.

**Browse the published site:** [booherbg.github.io/game-cheat-sheets](https://booherbg.github.io/game-cheat-sheets/)

**About this project:** See the [README](https://github.com/booherbg/game-cheat-sheets/blob/main/README.md) for setup and build instructions. Source code: [GitHub](https://github.com/booherbg/game-cheat-sheets)

**Docs:** [Prompts](docs/prompts.html) (prompts used to build the site) and [History](docs/history.html) (trick-taking and euchre history, with sources).
